مشروع لوحة تحكم تفاعلية متقدمة لتحليل بيانات الناتج المحلي (GDP) عالمياً، تم تطويره باستخدام MapLibre GL JS.
يعتمد المشروع على دمج مصدرين مختلفين للبيانات في وقت التشغيل:
1. الطبقة الجغرافية (GeoJSON): تحتوي على الحدود الهندسية لكل دول العالم.
2. بيانات الـ API (World Bank): تحتوي على القيم الاقتصادية (GDP) لكل دولة.
طريقة الربط:
 يتم جلب البيانات من API البنك الدولي عبر دالة ⁠fetchGDPData⁠ باستخدام ⁠async/await⁠.
 يتم استخدام ISO_A3 (رمز الدولة المكون من 3 أحرف) كـ "مفتاح فريد مشترك" (Unique Key).
 يتم عمل ⁠Mapping⁠ لهذه القيم داخل كائن (Object) في الجافاسكريبت، ثم نستخدم ⁠map.setPaintProperty⁠ مع تعبيرات (Expressions) مثل ⁠['match', ['get', 'ISO3166-1-Alpha-3'], ...]⁠ لربط قيمة الـ GDP باللون المناسب على الخريطة ديناميكياً.
📊 لوحة المؤشرات (Legend)
تم إضافة لوحة توضيحية (Legend) تعكس تدرج الألوان المعتمد:
 أحمر غامق: اقتصادات ضخمة (> 1 ترليون دولار).
 أحمر متوسط: اقتصادات متوسطة.
 ألوان فاتحة: اقتصادات ناشئة أو منخفضة.
 التقنيات المستخدمة
 MapLibre GL JS: للمحرك الخرائطي.
 World Bank API: للبيانات الاقتصادية (Indicator: ⁠NY.GDP.MKTP.CD⁠).
 OpenWeatherMap API: لجلب درجات الحرارة اللحظية للعواصم.
 Chart.js: لعرض البيانات التاريخية لدولة محددة.
💡 كيفية الاستخدام
1. Slider السنة: لتغيير السنة وملاحظة التغير في ألوان الدول في الخريطة وقيم الـ Top 5.
2. التفاعل مع الدول: عند النقر على أي دولة، تظهر نافذة (Popup) تحتوي على اسم العاصمة ودرجة الحرارة الحالية، بالإضافة إلى تحديث الرسم البياني الجانبي.
